🔑 API Setup

This gallery uses two 100% free services with no credit card required. Follow the steps below — it takes about 5 minutes!

Step 1 · ImgBB API Key — Image Hosting

1. Go to imgbb.com and create a free account.
2. Visit api.imgbb.com and click "Get API key".
3. Copy the key and paste it below.

Step 2 · JSONbin — Photo Metadata Storage

1. Go to jsonbin.io and create a free account.
2. Click "+ Create Bin", paste [] as the content, and click Save.
3. Copy the Bin ID from the URL bar (it looks like 6649abc123…).
4. Go to your Dashboard → API Keys and copy your X-Master-Key.

Keys are stored only in your browser. They are sent directly to ImgBB and JSONbin — never to any other server.
